# Заполнение базы данных тестовыми данными

## Доступные скрипты

### 1. Базовое заполнение (малый набор данных)
```bash
npm run db:seed
```

Создает:
- 6 пользователей
- 4 связи (матча)
- Несколько сообщений
- 6 объявлений на доске
- 5 откликов

### 2. Расширенное заполнение (большой набор данных)
```bash
npm run db:seed-large
```

Создает:
- **100 пользователей** (родители и няни)
- **~50 связей** между пользователями
- **~200 сообщений** в чатах
- **30 объявлений** на доске
- **~50 откликов** на объявления
- **~80 реакций** (лайки/блокировки)
- **~40 отзывов** для нянь

## Тестовый аккаунт

После запуска `npm run db:seed-large` будет создан тестовый аккаунт:

- **Email:** `test@example.com`
- **Пароль:** `password123`

Используйте эти данные для входа в приложение.

## Важно

⚠️ **Скрипт `db:seed-large` полностью очищает базу данных перед заполнением!**

Все существующие данные будут удалены. Используйте этот скрипт только для тестирования и разработки.

## Что включает расширенное заполнение

### Пользователи
- Разнообразные имена и фамилии
- Случайные аватары (эмодзи)
- Разные города (Москва, Санкт-Петербург, и др.)
- Типы: родители (70%) и няни (30%)
- Случайные интересы и биографии
- Геолокация (координаты)
- Рейтинги и карма

### Связи и сообщения
- Случайные связи между пользователями
- Реалистичные диалоги
- Прочитанные и непрочитанные сообщения
- Разные временные метки

### Объявления
- Разные типы: поиск няни, предложение услуг, встречи, и др.
- Активные и закрытые объявления
- Счетчики просмотров
- Отклики от других пользователей

### Реакции и отзывы
- Лайки между пользователями
- Отзывы для нянь с рейтингами
- Реалистичные комментарии

## Примеры использования

### Первый запуск проекта
```bash
# Применить миграции
npx prisma migrate dev

# Заполнить базу большим набором данных
npm run db:seed-large

# Запустить приложение
npm run dev
```

### Сброс и повторное заполнение
```bash
# Просто запустите скрипт снова
npm run db:seed-large
```

### Просмотр данных
```bash
# Открыть Prisma Studio для просмотра БД
npx prisma studio
```

## Структура данных

Скрипт создает реалистичные данные с:
- Правильными связями между таблицами
- Временными метками в прошлом
- Разнообразием типов и статусов
- Случайным распределением параметров

Все данные генерируются случайным образом, но выглядят реалистично для демонстрации и тестирования приложения.
